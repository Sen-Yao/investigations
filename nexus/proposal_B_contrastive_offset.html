📋 核心思想

借鉴 Jinsong Chen NeurIPS 2024 论文，设计 Offset 专用的对比学习目标。

目标：

  • 增强同一节点不同采样的 Offset 表示一致性
  • 增强 Offset 表示的可分性
  • 提升异常检测性能

📐 对比学习基础

对比学习核心概念

基本框架

对比学习的目标：

拉近（正样本对）: 同一类别的样本在表示空间中靠近
推远（负样本对）: 不同类别的样本在表示空间中远离

损失函数（InfoNCE）：

L = -log( exp(sim(z_i, z_i+)/τ) / Σ_j exp(sim(z_i, z_j)/τ) )

其中：
- z_i: 锚点样本的表示
- z_i+: 正样本的表示
- z_j: 所有样本（包括正负）
- τ: 温度参数

关键设计问题

问题选择理由
正样本定义同一节点不同邻居采样同一节点的 Offset 应一致
负样本定义不同节点的 Offset不同节点的 Offset 应可分
表示层级Offset 向量本身 或 Transformer 输出需要实验验证
对比粒度Hop-level 或 Node-level需要实验验证

🔬 方法设计

方案 B1: Hop-level Contrastive Learning

思路

对每个 hop 的 Offset 进行对比学习。

实现

class HopLevelContrastiveLoss(nn.Module):
    def __init__(self, temperature=0.2):
        super().__init__()
        self.temperature = temperature
    
    def forward(self, offsets_1, offsets_2, node_ids):
        """
        offsets_1: [N, K, D] - 第一次采样的 Offset
        offsets_2: [N, K, D] - 第二次采样的 Offset（同一节点）
        node_ids: [N] - 节点 ID
        
        返回：对比损失
        """
        N, K, D = offsets_1.shape
        
        # 展平为 [N*K, D]
        z1 = offsets_1.view(N*K, D)
        z2 = offsets_2.view(N*K, D)
        
        # 归一化
        z1 = F.normalize(z1, dim=-1)
        z2 = F.normalize(z2, dim=-1)
        
        # 计算相似度矩阵 [N*K, N*K]
        sim_matrix = torch.mm(z1, z2.t()) / self.temperature
        
        # 正样本对：同一节点的同一 hop
        # 即 indices i 和 i 是正样本对
        labels = torch.arange(N*K, device=z1.device)
        
        # 交叉熵损失
        loss = F.cross_entropy(sim_matrix, labels)
        
        return loss

特点

  • 细粒度：每个 hop 独立对比
  • 计算高效：只需两次采样
  • 缺点：可能忽略 hop 间关系

方案 B2: Node-level Contrastive Learning

思路

将整个 Offset 序列聚合成节点表示，再进行对比。

实现

class NodeLevelContrastiveLoss(nn.Module):
    def __init__(self, temperature=0.2, hidden_dim=256):
        super().__init__()
        self.temperature = temperature
        self.projector = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim)
        )
    
    def forward(self, offset_sequences_1, offset_sequences_2, node_ids):
        """
        offset_sequences_1: [N, K, D] - 第一次采样的 Offset 序列
        offset_sequences_2: [N, K, D] - 第二次采样的 Offset 序列
        
        返回：对比损失
        """
        # 聚合 Offset 序列为节点表示
        # 方法1: 平均池化
        node_repr_1 = offset_sequences_1.mean(dim=1)  # [N, D]
        node_repr_2 = offset_sequences_2.mean(dim=1)  # [N, D]
        
        # 方法2: Attention 池化（可选）
        # node_repr_1 = self.attention_pool(offset_sequences_1)
        
        # 投影到对比空间
        z1 = self.projector(node_repr_1)
        z2 = self.projector(node_repr_2)
        
        # 归一化
        z1 = F.normalize(z1, dim=-1)
        z2 = F.normalize(z2, dim=-1)
        
        # 计算相似度
        sim_matrix = torch.mm(z1, z2.t()) / self.temperature
        
        # 正样本：对角线
        labels = torch.arange(len(node_ids), device=z1.device)
        
        loss = F.cross_entropy(sim_matrix, labels)
        
        return loss

特点

  • 全局视角：考虑整个 Offset 序列
  • 灵活聚合：可用不同池化方法
  • 缺点：可能丢失 hop 细节

方案 B3: Supervised Contrastive Learning（推荐）

思路

利用标签信息，正常节点互为正样本，异常节点互为正样本。

实现

class SupervisedContrastiveLoss(nn.Module):
    def __init__(self, temperature=0.2):
        super().__init__()
        self.temperature = temperature
    
    def forward(self, node_reprs, labels):
        """
        node_reprs: [N, D] - 节点表示（从 Offset 聚合）
        labels: [N] - 0=正常，1=异常
        
        返回：对比损失
        """
        # 归一化
        z = F.normalize(node_reprs, dim=-1)
        
        # 计算相似度矩阵
        sim_matrix = torch.mm(z, z.t()) / self.temperature
        
        # 构建正样本掩码
        # 同标签 = 正样本
        labels = labels.view(-1, 1)
        positive_mask = (labels == labels.t()).float()
        
        # 移除对角线（自身不是正样本）
        positive_mask.fill_diagonal_(0)
        
        # 负样本掩码
        negative_mask = 1 - positive_mask
        negative_mask.fill_diagonal_(0)
        
        # Supervised Contrastive Loss
        # 正样本：同标签
        # 负样本：不同标签
        
        exp_sim = torch.exp(sim_matrix)
        
        # 正样本相似度
        pos_sim = (exp_sim * positive_mask).sum(dim=1)
        
        # 所有相似度
        all_sim = exp_sim.sum(dim=1) - exp_sim.diag()  # 减去自身
        
        # 损失
        loss = -torch.log(pos_sim / all_sim + 1e-8).mean()
        
        return loss

特点

  • 利用标签：充分利用监督信息
  • 聚类效果：正常/异常各自聚集
  • 适合半监督：只用有标签节点计算对比损失
