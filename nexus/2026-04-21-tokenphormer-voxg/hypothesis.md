# Hypotheses

## H1
NDC 高并不意味着异常更“正常”，而意味着异常节点与其邻域在 Delta 演化上存在更强的一致性。

## H2
ANR 高说明异常在图中往往形成局部社区，因此异常检测不应只建模单点偏离，还应建模社区内部一致模式。

## H3
Delta token 主要编码传播过程的变化量，而 neighborhood prototype token 可以编码“节点与邻域平均传播模式的关系”。

## H4
Community-aware token 可以作为 ANR 的无监督近似代理，帮助模型表达局部异常社区一致性。

## H5
VoxG 的下一代 token design 可以从单一 hop/delta 表示，扩展为 anomaly-aware multi-token family。
