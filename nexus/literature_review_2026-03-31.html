📖 详细文献分析

1. GDN: Graph Deviation Network

AAAI 2021

论文标题：Graph Neural Network-Based Anomaly Detection in Multivariate Time Series

作者：Ailin Deng, Bryan Hooi

链接：https://bhooi.github.io/papers/gdn_aaai2021.pdf

核心问题

如何在多变量时间序列中检测异常事件（如系统故障、攻击）？

方法框架

组件 功能 实现方式
Sensor Embedding 捕获每个传感器的独特特征 学习每个传感器的嵌入向量
Graph Structure Learning 学习传感器之间的依赖关系 注意力机制学习边权重
Graph Attention Forecasting 预测传感器未来行为 基于邻居传感器的注意力预测
Graph Deviation Scoring 识别偏离正常模式的异常 计算预测值与实际值的偏差

关键创新

  • 可解释性：通过学习到的图结构和嵌入，可以推断异常的根本原因
  • 自适应图学习：自动学习传感器间的依赖关系，无需先验知识
  • 预测-偏差范式：先学习正常行为模式，再通过偏差识别异常

与 Hop-Offset 的关系

核心相似性：
  • Deviation Scoring ≈ Offset 计算：都是计算偏离"正常"的量
  • GDN 计算的是预测值与实际值的偏差
  • Hop-Offset 计算的是当前 Hop 与基线的偏移
  • 两者都通过偏差量来识别异常

2. Token-based Graph Anomaly Detection

Survey

核心概念

"Token-based" 指将图的组成部分（节点、边、子图、属性、时序序列）表示为离散单元或 "tokens"，然后输入深度学习模型处理。

Token 类型与用途

Token 类型 表示内容 应用场景
节点级 Tokens 节点特征向量 节点异常检测
边级 Tokens 边编码、边属性 边异常检测、动态图
子图级 Tokens 局部结构表示 子图异常、欺诈团伙检测
时序 Tokens 时间步特征 动态图异常、时间序列

代表模型

模型 应用场景 Token 化方式 核心架构
GMRT 交通异常检测 图结构数据 → patches Multi-head Self-Attention
AET 动态图异常检测 边编码 → tokens Transformer Module
DeFi Fraud Detection 金融欺诈检测 交易序列 → tokens GNN + Transformer
Log Anomaly 日志异常检测 日志序列 → tokens (BERT-style) BERT Encoder

关键技术

  • Self-Attention：捕获 tokens 之间的长程依赖
  • Multi-head Attention：多分辨率特征提取
  • Positional Encoding：保留位置/顺序信息
  • Special Tokens：[CLS], [SEP], [PAD] 用于序列级表示

与 Hop-Offset 的关系

核心相似性：
  • Hop Vector ≈ Token：每个 Hop 的特征向量可视为一个 token
  • Delta/Offset ≈ Token 差分：类似于 NLP 中的差分表示
  • 多跳聚合 ≈ Multi-head Attention：都是从多个位置聚合信息
潜在方向：
  • 将 Hop-Offset 向量作为特殊 tokens 输入 Transformer
  • 探索 Hop-Offset 的 Positional Encoding 效果
  • 对比 Hop-Offset 与标准 Token 表示的差异

3. Contrastive Learning for GAD

arXiv 2022-2025

核心思想

通过对比学习增强正常图/节点的表示，使异常样本在表示空间中更加可分。

代表论文详解

3.1 GLADC (Scientific Reports 2022)

论文标题：Deep graph level anomaly detection with contrastive learning

链接：https://www.nature.com/articles/s41598-022-22086-3

模块 功能 技术
Dual-graph Encoder 学习节点级和图级表示 GCN Autoencoder + 扰动编码器
Contrastive Learning 增强图表示 最大化正样本对的相似度
Graph Anomaly Detection 识别异常图 重构误差作为异常分数

关键创新：

  • 无需数据增强，通过扰动编码器构建对比样本
  • 联合优化节点级和图级表示
  • 端到端框架，优于两阶段方法
3.2 EAGLE (arXiv 2025)

论文标题：EAGLE: Contrastive Learning for Efficient Graph Anomaly Detection

核心思想：在异构图上通过对比学习检测异常节点，对比正常节点与异常节点到局部上下文的距离。

3.3 Rethinking Contrastive Learning (arXiv 2025)

论文标题：Rethinking Contrastive Learning in Graph Anomaly Detection: A Clean-View Perspective

核心贡献：

  • 提出 Clean-View 视角，解决对比学习中异常样本污染问题
  • 多尺度异常感知模块
  • Node-Subgraph (NS) Contrast + Node-Node (NN) Contrast

对比学习的关键技术

技术 作用 在 GAD 中的应用
正样本构建 定义相似样本对 同节点不同视图、邻居节点
负样本采样 定义不同样本对 不同节点、不同图
InfoNCE Loss 优化目标 最大化正样本相似度，最小化负样本相似度
Temperature τ 控制分布锐度 通常 0.1-0.5

与当前研究的关系

与 Fisher 可分性的关系：
  • 对比学习目标：增大类间距离，减小的类内距离
  • Fisher 可分性目标：最大化类间散度/类内散度
  • 两者目标一致，可以结合使用
潜在方向：
  • 使用对比学习预训练 Hop-Offset 表示
  • 设计 Hop-Offset 特有的对比学习目标
  • 结合 Fisher 可分性分析优化对比学习

📊 方法对比总结

方法 核心思想 Token 表示 异常检测方式 与 Hop-Offset 关系
GDN 学习传感器依赖图，预测行为 传感器嵌入 Graph Deviation Scoring ⭐⭐⭐ Deviation ≈ Offset
Token-based GAD 将图元素表示为 tokens 节点/边/子图 tokens Transformer 异常检测 ⭐⭐ Hop Vector ≈ Token
GLADC 对比学习增强图表示 节点/图嵌入 重构误差 ⭐⭐ 表示增强
EAGLE 异构图对比学习 节点嵌入 距离异常分数 ⭐ 可扩展到 Hop-Offset
