## 本质问题分析

### 为什么 GT 学不到 Delta？

| 问题 | 说明 |
|------|------|
| **标准注意力设计** | QK^T 计算的是**相似度**（点积）|
| **Delta 的本质** | Hop_{k+1} - Hop_k 是**差异**（减法）|
| **矛盾** | 注意力机制没有减法操作！|

### 数学分析

```
标准注意力：
Attention(Q, K, V) = softmax(QK^T / √d) V

问题：
- QK^T = 点积 = 余弦相似度 × 模长
- 衡量有多相似，不是有多不同
- Delta 是减法操作，标准注意力没有！
```

## 结论

### 放弃显式 Delta 构建

| 原因 | 说明 |
|------|------|
| **不优雅** | trick 感强，不够创新 |
| **数据集依赖** | 效果不稳定 |

### 推荐新方向

| 方案 | 说明 |
|------|------|
| **Hop-物理意义注入注意力** | 让注意力机制知道 Hop 的因果结构 |
| **优雅性** | ⭐⭐⭐ 高（理论清晰，最小修改）|
