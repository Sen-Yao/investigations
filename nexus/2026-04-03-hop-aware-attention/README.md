# Hop-Aware Attention 设计与验证

> 验证 Hop-物理意义注入注意力的可行性，解决 GT 无法学习 Delta 的本质问题

---

## 背景

### 核心发现

| 发现 | 说明 |
|------|------|
| **GT 学习能力孱弱** | 标准注意力 QK^T 计算相似度，没有减法 |
| **Delta 是减法操作** | Transformer 无法自动学习 Delta（Photo R² = -0.15）|
| **显式 Delta 不优雅** | trick 感强，不够本质创新 |

### LLM 咨询结果

GLM-5.1 建议采用**相对距离偏置**方案：
- hop_bias 基于 Hop 相对距离（hop[j] - hop[i]）
- 物理意义：相邻层比较，捕捉异常
