## 可能原因分析

### 为什么平滑正则化有害？

1. **特征已足够平滑**
   - Photo 数据集的特征分布可能已经比较平滑
   - 额外的平滑约束是冗余的

2. **干扰异常模式学习**
   - 异常检测需要识别"不正常"的模式
   - 强制平滑可能抹平了有价值的异常信号

3. **与 Hop Token 设计冲突**
   - Hop Token 本身已经包含了邻域聚合
   - 再加平滑约束可能过度平滑
