# Insights: 平滑正则化效果分析

## 🔍 Smooth Loss 实现分析（重要发现！）

### 实现代码

```python
# L_smooth = Σ_{k=0}^{K-1} ||z_k - z_{k+1}||² / σ²
# 取所有相邻层输出的 MSE 均值
smooth_loss = torch.stack([torch.mean((layer_outputs[k+1] - layer_outputs[k])**2) 
                           for k in range(len(layer_outputs)-1)]).mean() / sigma_sq
```

### 关键洞察：这是层间平滑，不是节点间平滑！

| GLM-5.1 假设 | 实际实现 |
|-------------|----------|
| 节点间平滑 \|h_i - h_j\|² | ❌ 不是这个 |
| 层间平滑 \|z_k - z_{k+1}\|² | ✅ 实际实现 |

### 为什么层间平滑更有害？

| 层级学习 | Smooth Loss 效果 |
|---------|-----------------|
| 浅层：局部特征 | 被强制与深层相似 |
| 深层：全局语义 | 被强制与浅层相似 |
| **结果** | **所有层输出趋同 → 丢失层级信息** |

**Transformer 的层级特征学习对异常检测至关重要，层间平滑抹杀了这种能力！**

## 核心洞察

### 1. 层间平滑 vs 节点间平滑

| 类型 | 公式 | 作用 | 对异常检测的影响 |
|------|------|------|-----------------|
| **节点间平滑** | Σ \|h_i - h_j\|² | 拉平邻居表示 | 可能抹杀异常信号 |
| **层间平滑** | Σ \|z_k - z_{k+1}\|² | 拉平层级表示 | **抹杀层级特征学习** |

**当前实现是层间平滑，对 Transformer 更有害！**

### 2. 与昨天发现的关联

**昨天的发现**: GT 学不到 Delta（差异信息）

**今天的发现**: 层间平滑损害 Transformer 层级学习

**共同点**: 都涉及 **Transformer 的表达能力受限**
- 标准注意力没有减法 → 学不到 Delta
- 层间平滑 → 层级特征被抹平

### 3. Transformer 层级特征的重要性

```
Layer 1: 局部结构特征（如邻居度分布）
Layer 2: 中层语义（如子图模式）
Layer 3: 全局语义（如节点角色）

层间平滑 → 所有层输出趋同 → 丢失多层信息
```
