# 双流架构理论基础验证

> 验证"专用通道保留 Delta 信息 > Attention"的理论假设

---

## 背景

### 已验证的发现

| 发现 | 数据集 | 结果 |
|------|--------|------|
| concat 最佳但仍不足 | Photo | AUC 0.8777 vs VecGAD 0.8960 (-1.8%) |
| Attention 摧毁 Delta | Photo | 线性 R²=0.80, 探测 R²=-0.15 |
| hop_bias 失败 | Photo | AUC 0.7943 (干扰异常感知) |

### GLM-5.1 推荐方向

**双流架构**：给差异信息开一条专用通道

```
Stream S: 标准 Attention（处理结构模式）
Stream D: MLP（处理 Delta，不经点积）
Output = Gate(S, D)
```
