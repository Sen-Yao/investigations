## V1 记录错误分析

### 问题

V1 记录的 0.8966 AUC 被错误归因于 Photo 数据集。

### 根本原因

1.  同时运行 Amazon 和 Photo
2. 0.8966 来自 Amazon 数据集
3. 记录时未仔细核对数据集来源

### 更正

| 版本 | 记录 AUC | 实际数据集 |
|------|---------|-----------|
| V1 (错误) | 0.8966 | ~~Photo~~ → Amazon |
| V1 (真实) | 0.8727 | Photo (单 seed) |
| V3 | 0.8316±0.0132 | Photo (5-seed) |
