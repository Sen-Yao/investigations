📋 核心问题

Offset 向量的"方向"是否携带与异常检测相关的语义信息？

具体来说：

  • 正常节点的 Offset 方向是否聚集？
  • 异常节点的 Offset 方向是否偏离正常方向？
  • 偏离程度是否与异常程度相关？

📐 Offset 方向的数学定义

Offset 向量

对于节点 v，其 k-hop 的 Offset 向量定义为：

Offset_k(v) = Feature(Hop_k(v)) - Feature(Hop_0(v))

其中：
- Feature(Hop_0(v)) = 节点 v 自身的特征
- Feature(Hop_k(v)) = k-hop 邻居的聚合特征
- Offset_k(v) ∈ R^D (D 为特征维度)

方向向量

方向向量 = Offset / ||Offset||_2

归一化后，只保留方向信息，丢失大小信息。

方向向量 ∈ S^{D-1} (D-1 维单位球面上的点)

方向相似度

两个方向向量的相似度 = 余弦相似度

sim(direction_i, direction_j) = (direction_i · direction_j) / (||direction_i|| × ||direction_j||)

范围：[-1, 1]
- 1 表示完全相同方向
- 0 表示正交
- -1 表示相反方向

🧠 理论假设

假设 1: 聚集假设

内容：正常节点的 Offset 方向聚集在某个"正常方向"附近。

直觉：正常节点具有相似的特征变化模式。

验证指标：正常节点方向向量的方差（越小越支持假设）

假设 2: 偏离假设

内容：异常节点的 Offset 方向偏离"正常方向"。

直觉：异常节点的特征变化模式与正常节点不同。

验证指标：异常节点方向与正常中心的夹角（越大越支持假设）

假设 3: 相关假设

内容：方向偏离程度与异常程度正相关。

直觉：越异常的节点，其 Offset 方向越偏离正常方向。

验证指标：方向偏离度与异常分数的 Spearman 相关系数

🔬 实验设计

实验 1: 方向分布可视化

目的

直观观察正常/异常节点的 Offset 方向分布差异。

方法

1. 提取所有节点的 Offset 方向向量
2. 使用 t-SNE / UMAP 降维到 2D
3. 按标签（正常/异常）着色散点图
4. 使用核密度估计 (KDE) 绘制分布轮廓

预期观察

  • 支持假设：正常节点聚集，异常节点分散或形成独立簇
  • 不支持假设：两类节点分布重叠，无明显差异

实验 2: 方向聚集度分析

目的

量化正常/异常节点的方向聚集程度。

方法

1. 计算正常节点的"方向中心"（平均方向向量）
   center_normal = normalize(mean(directions_normal))

2. 计算每个节点与中心的方向相似度
   similarity_i = cosine_sim(direction_i, center_normal)

3. 比较正常/异常节点的相似度分布
   - 统计量：均值、方差、分位数
   - 假设检验：Mann-Whitney U test

评估指标

指标计算方式预期结果
正常节点平均相似度mean(sim_normal)接近 1
异常节点平均相似度mean(sim_anomaly)显著低于正常
方向方差比var(anomaly) / var(normal)> 1

实验 3: 方向-异常相关性分析

目的

验证方向偏离度与异常分数的相关性。

方法

1. 计算方向偏离度
   deviation_i = 1 - cosine_sim(direction_i, center_normal)

2. 获取异常分数（从模型输出或标签）
   anomaly_score_i = model_output(node_i) 或 label_i

3. 计算相关性
   - Spearman 相关系数
   - Kendall's tau
   - Pearson 相关系数

评估指标

相关系数范围预期值
Spearman ρ[-1, 1]> 0.3 (中等相关)
Kendall's τ[-1, 1]> 0.2

实验 4: Hop-wise 方向一致性分析

目的

分析不同 hop 的 Offset 方向是否一致。

方法

对于每个节点，有 K 个 Offset 向量：[Offset_1, Offset_2, ..., Offset_K]

方向一致性 = 方向向量之间的平均相似度

consistency = mean(cosine_sim(Offset_i, Offset_j)) for all i ≠ j

预期发现

  • 正常节点：不同 hop 方向可能一致（特征变化稳定）
  • 异常节点：不同 hop 方向可能不一致（特征变化不稳定）

📊 分析脚本框架

# offset_direction_analysis.py

import numpy as np
from sklearn.manifold import TSNE
import matplotlib.pyplot as plt
from scipy import stats

class OffsetDirectionAnalyzer:
    def __init__(self, offsets, labels):
        """
        offsets: [N, K, D] - N个节点，K个hop，D维特征
        labels: [N] - 0=正常，1=异常
        """
        self.offsets = offsets
        self.labels = labels
        self.directions = self._normalize(offsets)
    
    def _normalize(self, vectors):
        """归一化为方向向量"""
        norms = np.linalg.norm(vectors, axis=-1, keepdims=True)
        return vectors / (norms + 1e-8)
    
    def visualize_distribution(self, save_path):
        """可视化方向分布"""
        # 展平方向向量
        directions_flat = self.directions.reshape(-1, self.directions.shape[-1])
        
        # t-SNE 降维
        tsne = TSNE(n_components=2, random_state=42)
        directions_2d = tsne.fit_transform(directions_flat)
        
        # 按标签着色
        plt.figure(figsize=(10, 8))
        normal_mask = self.labels == 0
        anomaly_mask = self.labels == 1
        
        plt.scatter(directions_2d[normal_mask, 0], directions_2d[normal_mask, 1], 
                   c='blue', alpha=0.5, label='Normal')
        plt.scatter(directions_2d[anomaly_mask, 0], directions_2d[anomaly_mask, 1], 
                   c='red', alpha=0.5, label='Anomaly')
        
        plt.legend()
        plt.title('Offset Direction Distribution')
        plt.savefig(save_path)
        plt.close()
        
        return directions_2d
    
    def compute_direction_center(self, subset='normal'):
        """计算方向中心"""
        if subset == 'normal':
            mask = self.labels == 0
        else:
            mask = self.labels == 1
        
        directions_subset = self.directions[mask]
        center = np.mean(directions_subset, axis=(0, 1))  # 平均方向
        return center / np.linalg.norm(center)  # 归一化
    
    def compute_similarity_distribution(self):
        """计算与正常中心的相似度分布"""
        center = self.compute_direction_center('normal')
        
        similarities = np.einsum('nkd,d->nk', self.directions, center)
        
        normal_sims = similarities[self.labels == 0].flatten()
        anomaly_sims = similarities[self.labels == 1].flatten()
        
        return normal_sims, anomaly_sims
    
    def statistical_test(self):
        """统计检验"""
        normal_sims, anomaly_sims = self.compute_similarity_distribution()
        
        # Mann-Whitney U test
        statistic, p_value = stats.mannwhitneyu(normal_sims, anomaly_sims, alternative='greater')
        
        return {
            'normal_mean': np.mean(normal_sims),
            'normal_std': np.std(normal_sims),
            'anomaly_mean': np.mean(anomaly_sims),
            'anomaly_std': np.std(anomaly_sims),
            'mann_whitney_u': statistic,
            'p_value': p_value
        }
    
    def correlation_with_anomaly(self, anomaly_scores):
        """与异常分数的相关性"""
        center = self.compute_direction_center('normal')
        deviations = 1 - np.einsum('nkd,d->nk', self.directions, center)
        deviations = deviations.mean(axis=1)  # 平均偏离度
        
        # Spearman 相关
        rho, p = stats.spearmanr(deviations, anomaly_scores)
        
        return {'spearman_rho': rho, 'p_value': p}
