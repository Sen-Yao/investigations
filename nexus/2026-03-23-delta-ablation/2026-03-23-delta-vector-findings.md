# Delta向量异常检测能力分析报告

**报告日期**: 2026-03-23  
**研究背景**: 探索Delta向量（不取L2范数）作为异常检测Token序列的可行性

---

## 一、核心发现

### 1.1 Delta向量 vs Delta范数性能对比（tolokers数据集）

| 方法 | AUC | AP | 说明 |
|------|-----|-----|------|
| **Delta向量（flatten）** | **0.7070** | **0.3633** | 展平所有hop和特征维度 |
| Delta向量（mean） | 0.6470 | 0.3106 | 对hop维度取平均 |
| Delta向量（sum） | 0.6470 | 0.3106 | 对hop维度求和 |
| Delta向量（last） | 0.6389 | 0.3053 | 只用最后一个hop |
| Delta向量（max） | 0.6250 | 0.3083 | 对hop维度取最大 |
| **Delta范数（基线）** | **0.5342** | **0.2631** | 当前方法的L2范数 |

**关键发现**：Delta向量（flatten方法）比Delta范数提升 **+0.17 AUC**！

### 1.2 各hop单独评估

**观察**：
- 早期hop的Delta向量效果更好
- 所有hop的向量形式都优于范数形式

### 1.3 信息冗余度分析

| 指标 | 数值 |
|------|------|
| 平均相关性 | **0.9944** |
| 最大相关性 | 0.9980 |
| 最小相关性 | 0.9851 |

**重要发现**：Delta向量与原始Token高度相关（相关性>0.99），这意味着：
- Delta向量虽然检测能力强，但与原始Token信息高度重叠
- 作为独立特征使用可能存在冗余
- 需要考虑如何有效利用这种信息

## 二、理论分析

### 2.1 为何Delta向量比Delta范数更有效？

| 特性 | Delta向量 | Delta范数 |
|------|-----------|-----------|
| 信息保留 | 完整保留每个特征维度的变化方向和幅度 | 只保留总体变化幅度 |
| 维度 | [N, K, D] | [N, K] |
| 区分能力 | 可学习各维度的不同权重 | 所有维度等权重聚合 |

**关键洞察**：不同特征维度的变化对异常检测的贡献不同：
- tolokers数据集特征1的AUC=0.66，特征7的AUC=0.56
- Delta范数将所有特征等权重聚合，损失了维度级别的区分信息
- Delta向量允许模型学习各维度的重要性权重

### 2.2 信息冗余度高的原因

Delta向量与原始Token高度相关（0.99+），这是因为：
1. **传播过程的性质**：PPR传播是平滑过程，相邻hop特征高度相关
2. **Delta的定义**：$\delta = X^{(k+1)} - X^{(k)}$，与$X^{(k)}$和$X^{(k+1)}$都相关
3. **低通滤波效应**：图传播本质上是低通滤波，导致特征趋同

## 三、对PromptGAD的改进建议

### 3.1 推荐方向：Delta向量作为辅助Token

虽然Delta向量与原始Token高度相关，但其强大的异常检测能力（AUC 0.707）不容忽视。

**方案**：将Delta向量作为辅助Token序列，与原始Token并行处理

```python
# 原始Token: [N, K+1, D]
# Delta向量: [N, K, D]

delta_tokens = node_tokens[:, 1:] - node_tokens[:, :-1]  # [N, K, D]

# 方案1: 拼接为扩展序列
extended_tokens = torch.cat([node_tokens, delta_tokens], dim=1)  # [N, 2K+1, D]

# 方案2: 残差增强
enhanced_tokens = node_tokens[:, :-1] + alpha * delta_tokens  # [N, K, D]
```

### 3.2 关键优势

1. **无需知道δ比值方向**：Delta向量本身就包含方向信息
2. **半监督友好**：不需要异常节点标签
3. **简单实现**：只需修改输入，不改变模型架构

### 3.3 预期效果

基于tolokers的实验结果，预计：
- Delta向量特征可提供额外0.15+ AUC的检测能力
- 但需要处理与原始Token的信息冗余问题

## 五、结论

**Delta向量（不取L2范数）作为异常检测特征是有效的**，在tolokers数据集上比Delta范数提升了+0.17 AUC。

但Delta向量与原始Token高度相关（相关性>0.99），建议：
1. 不要单独使用Delta向量替代原始Token
2. 可以作为辅助信息增强异常检测能力
3. 需要设计融合机制处理信息冗余
